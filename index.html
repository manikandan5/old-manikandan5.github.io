Some of the projects that I worked on

Yelp Dataset Challenge

Predicted the category of the restaurant using the review, tip and business information available in the YELP Dataset. (Java)

Weather Predictor

A fault tolerant prototype, built on Service Oriented Architecture. Deployed the microservices on Amazon EC2 instances with the database hosted on RDS. These services interact through REST calls. This features continuous integration, continuous deployment and load balancing.
(Java, Python, JavaScript)

Scene Classification

Scene classification is a very interesting problem of automatically labeling an image among a set of semantic categories. To achieve this, we had implemented a Bag Of Words model and used Cornell's SVMLight to train and classify the data. (C++)

Part of Speech tagger

Program which classifies the given sentence into its respective part of speech using Bayesian network models and Natural Language Processing techniques. (Python)

Operating System

XINU OS on a Beagle Bone Black box with process synchronization, basic memory management, networking and simple file systems. (C)

Stereo images

The program creates a stereo image from the normal image and the disparity map from the same view. If a seed image is given instead of a disparity map is not given, it tries to estimate the disparity using Markov Random Field energy minimization. (C++)

Object detection

The project classifies the images into a pre-defined class. We have compared the results found using a simple baseline(subsampled pixel points), Principal Component Analysis, Haar-like features, Bag Of Words model and a Convolutional Neural Network. (C++)

Image Matching, Stitching and Warping

The program finds the SIFT points in the given image. Then it compares the similarity of two images, ranks the images in a dataset based on the similarity to the query image. Also the program tries to project the given image in the same projection as another image. (C++)

Optical Music Note Recognition

The code finds the edges on the Musical note images first by performing convolution with a sobel filter. Then it tries to find the coordinates of the notes occur using template matching. It finds the location of staff lines using Hough transform. With this data, it estimates the correct note. (C++)

Rameses Player

Rameses is a two player game played on a board of NxN. It is similar to Tic-Tac-Toe. But instead of X's and O's, you have just X. When the player hits N X's in a line, player loses. It is the opposite of Tic-Tac-Toe. This is an AI player which tries to find the optimal play using Iterative Deepening
(Python)

Zacate Player

Zacate is traditional dice game played in Latin America. It is a game of chance. The rules of the game can be found in the github repository. The code here is an AI program which takes certain decisions to maximize the score. This program competed against other programs and came 4th in a class of 80. (Python)

Image Orientation finder

The program tries to find the right orientation of the given image. It trains on the given dataset and classifies the orientation using K-nearest neighbors and Neural networks.
(Python)
